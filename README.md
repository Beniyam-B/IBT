# Full Stack Development Training

This repository serves as a record of my progress throughout my **Full Stack Development Training** at **IBT**.

It contains the exercises, practice files, assignments, and projects completed during the training. The repository is updated regularly as I learn new technologies and complete new modules.

## Purpose

The main objectives of this repository are to:

* Track my learning journey throughout the training.
* Practice using Git and GitHub for version control.
* Store exercises, assignments, and projects in an organized manner.
* Build a portfolio that demonstrates my growth as a full stack developer.

## Repository Structure

The repository is organized into folders based on the training modules or days. Each folder contains the relevant practice exercises, assignments, and supporting files for that section of the course.

As the training progresses, additional folders and projects will be added to reflect new topics and technologies.

## Technologies

Throughout this training, I will be working with a variety of technologies, including but not limited to:

* Git & GitHub
* Python
* HTML & CSS
* JavaScript
* React
* Node.js
* Express.js
* Databases
* REST APIs
* Additional full stack development tools and frameworks

This repository will continue to evolve as I progress through the Full Stack Development program and gain experience with new technologies and best practices.
